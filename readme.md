a simple, minimalist and aesthetic quotes site.

## Features
- Display random quotes
- Simple and clean user interface
- Lightweight and fast
- No dependencies required

Currently, all the quotes are added manually. In future updates, I plan to integrate an API to fetch quotes dynamically.

## Local Installation 
1. Clone the repository
2. Open `index.html` in your browser

## Usage
Simply open the application in a web browser to view random quotes.
